Pullback Playbook real app package

This is a proper installable PWA package.
Chrome will only show Install / Add to Home screen after these files are hosted as a website.

Fastest method on phone:
1. Download this zip.
2. Move it to Documents if Netlify Drop cannot see Downloads.
3. Go to app.netlify.com/drop in Chrome.
4. Upload the zip.
5. Open the generated site URL.
6. Chrome should show Install Pullback Playbook.
