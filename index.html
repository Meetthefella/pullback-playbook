How In-App Analysis Works
Each ticker card builds a structured setup payload with ticker, market status, saved checklist, notes, chart context, and your fixed risk rules.

The browser sends that payload to a lightweight serverless endpoint. The endpoint calls the OpenAI Responses API securely using your server-side `OPENAI_API_KEY`, then returns structured analysis back into the card.
